# 麻雀ビジョン - デザイン仕様書

## デザインコンセプト

- **スタイル：** Material Design 3 をベースに、Notionライクなミニマルさを加えた設計
- **カラー：** モノクロ基調。白・グレー・黒のみで構成。アクセントカラーは使わない
- **フォント：** システムフォント or Noto Sans JP（日本語対応重視）
- **雰囲気：** スタイリッシュ・清潔感・情報密度のバランス

## 麻雀牌デザイン（SVG）

### コンセプト
- シンプルなフラットデザイン
- 白背景に黒線・黒文字
- 牌の絵柄はアイコン的なミニマルSVG
- 角丸の白い長方形に絵柄

### 牌のサイズ

| 用途 | サイズ |
|------|--------|
| セレクターパレット | 40×56px |
| 手牌表示（大） | 48×68px |
| 符数内訳（小） | 32×44px |
| 結果表示（ミニ） | 24×34px |

### SVGスタイル

```
牌の外形：角丸4px、白背景(#FFFFFF)、グレーボーダー(#D4D4D4)、影あり
数字牌：中央に数字＋下部に絵柄（ドットや線のシンプルな図案）
字牌：中央に漢字（東南西北中發白）
赤ドラ：数字が赤色（#EF4444）→ 唯一の例外カラー
```

### 牌一覧とSVG図案方針

**萬子（m）：** 数字 + 「萬」文字（小さく下部）
**筒子（p）：** 数字 + ドットを円形配置
**索子（s）：** 数字 + 竹のような縦線を配置
**字牌（z）：** 漢字のみ大きく表示（東・南・西・北・中・發・白）

## コンポーネント設計

### TileSelector（牌選択パレット）
- グリッドレイアウト（10列）
- 牌をタップで手牌に追加
- 枚数上限（4枚/牌種）に達したらグレーアウト

### HandDisplay（手牌表示）
- 横スクロール可能な牌の列
- 各牌をタップで削除
- あがり牌は少し浮かせて区別

### SettingChip（設定チップ）
- Material Chip風（角丸ボーダー）
- 選択中は黒背景・白テキスト
- 非選択は白背景・黒テキスト

## インタラクション

| アクション | フィードバック |
|------------|----------------|
| 牌選択 | 軽いスケールアップ（1.05倍）+ 影 |
| 牌削除 | フェードアウト＋スライドアウト |
| 計算実行 | ボタンにローディングスピナー（黒） |
| 結果表示 | フェードイン + スライドアップ |
| 符数展開 | アコーディオンアニメーション |
| エラー | シェイクアニメーション |

---

## アニメーション

- 時間：150ms〜250ms
- イージング：`cubic-bezier(0.4, 0, 0.2, 1)`（Material Standard）
- 過度なアニメーションは避ける

## アイコン

- Material Symbols または Heroicons（アウトラインスタイル）
- 使用例：
  - 📷 → camera icon
  - ⚙️ → settings icon
  - ▼ → expand_more
  - ✕ → close

## アクセシビリティ

- タッチターゲット最小 44×44px
- フォーカスリング表示（キーボード操作時）
- aria-label 付与（スクリーンリーダー対応）
- コントラスト比 4.5:1 以上
